# Agentic AI Resume Screening & Recommendation System

## Overview
This project implements a lightweight Agentic AI system that autonomously evaluates resumes against job role requirements and provides intelligent hiring recommendations.

## Agentic Workflow
- Perception: Resume text ingestion
- Reasoning: Skill matching and scoring logic
- Action: Suitability score and final recommendation

## Technologies Used
- Python
- Basic NLP
- Rule-based reasoning

## Use Case
Automates resume screening for recruiters and HR teams.

## Future Enhancements
- LLM-based reasoning
- Multi-agent collaboration
- PDF resume ingestion
